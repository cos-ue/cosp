Die Datei ist nicht direkt durch den Nutzer aufrufbar, dies wird durch folgenden Code-Ausschnitt sichergestellt:
\begin{lstlisting}[language=php]
if (!defined('NICE_PROJECT')) {
	die('Permission denied.');
}
\end{lstlisting}
