# COSP - Citizen Open Science Platform

This is the core of our citizen science platform.

As a central part this provides:
- user authentication and management
- management of pictures
- management of texts
- management of modules

To use this features in a module there are an APIs implemented. The usage of those can be found in the Documentation. 
Currently main part of documentation si only available in German.
Furthermore it is also capabale of providing some sort of basic statistics.
